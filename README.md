# Анализ товаров продавцов

## Описание проекта
Проект посвящен анализу данных о товарах продавцов на маркетплейсе. В ходе исследования проводится обработка исходных данных, расчет дополнительных метрик и показателей для формирования комплексного отчета о товарах.

## Исследуемый набор данных
Набор данных содержит информацию о товарах продавцов, включая:
- Базовые характеристики товаров (название, категория, бренд)
- Информацию о продавцах и складах
- Показатели продаж и доступности
- Рейтинги и статистику продаж

Данные содержат 15 основных полей, на основе которых рассчитываются дополнительные бизнес-метрики.

## Цели исследования
1. Обогащение исходных данных расчетными показателями
2. Анализ эффективности продаж товаров
3. Расчет потенциальной выручки и прогнозов продаж
4. Оценка рейтингов товаров относительно друг друга

## Основные расчетные показатели
- **returned_items_count** - количество возвращенных товаров
- **potential_revenue** - потенциальный доход от остатков
- **total_revenue** - доход с учетом возвратов
- **avg_daily_sales** - средние дневные продажи
- **days_to_sold** - прогноз дней до распродажи остатков
- **item_rate_percent** - процентиль рейтинга товара
